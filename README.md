# pa33wd

a simple tool to help save you passwords, encrypt by aes.

**usage:**

 > save your password:
 >> pa33.exe xxx.com xxx xxx

 > get your password:
 >> pa33.exe xxx.com xxx
 >> pa33.exe xxx.com

 > import password file(.csv)
 >> pa33.exe -i xxx.csv. (only csv file)

 > delete one password
 >> pa33.exe -d xxx.com xxx
